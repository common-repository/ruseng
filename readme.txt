== Description ==

The plugin makes transliteration of Russian links into English.

== Installation ==

1. Go to <strong>Plugins</strong> > <strong>Add New</strong>.
2. Search for <strong>RusEng</strong>.
3. Press <strong>Install Now</strong> and then <strong>Activate</strong>.
